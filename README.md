## 📥 Getting started

```bash

    # Clone repository
    $ git clone https://github.com/israelsantana01/happy.git

    # Select server folder
    $ cd happy/backend/

    # Start database
    $ npm start

    # Select web folder
    $ cd happy/web/

    # Start app in your browser
    $ npm start
```
